# 

## 进程与线程

::: tip 描述

进程是操作系统**分配资源**的基本单位，线程是操作**系统调度**的基本单位，线程可以看做轻量级的进程。

:::

**不同点：**

1. 一个进程中至少包含一个线程，线程不能包含进程，不能单独存在，必须由进程启动和管理
2. 多个进程间不能共享资源，都有自己私有的程序计数器（记录执行指令地址）,线程可以共享同一个进程间的资源
3. 上下文切换：进程慢，线程快（从一个切换到另一个）
4. 操纵者不同：进程操纵者是操作系统，线程操纵者一般为程序员
