## 实现一个元素的宽高比
```css
.wrapper {
    width: 100%;
    height: 0;
    padding-top: 56.25%; /* 16:9 宽高比 */
    position: relative;
}

.content {
    position: absolute;
    top: 0;
    left: 0;
    bottom: 0;
    right: 0;
}
```

**维持正方形**: 宽度发生改变时，利用伪元素撑开高度，从而保持正方形。

```css
.square::after {
  content: "";
  display: block;
  padding-bottom: 100%;
}
```
